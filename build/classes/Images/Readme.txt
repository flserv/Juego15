Paquete Images
El paquete "Images" contiene cuatro archivos de imagen.
El archivo "negro.png", que la aplicación utiliza cuando se crea un nuevo archivo de puzle.
Los archivos "SelfPortraitInOwlSEye.png" y "AfanDeProtagonismo.png" contienen dos imágenes que han servico para crear los archivos "SelfPortraitInOwlSEye.pzl" y "AfanDeProtagonismo.pzl".
El archivo "OcasoEnSonOliva.png" se incluye a modo de ejemplo de imagen cuadrada susceptible de ser utilizada para crear un nuevo puzle.
